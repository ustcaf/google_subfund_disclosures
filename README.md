# USTCAF 管理 Google Subfund 之信息披露

## 历史与致谢

Google 子基金由刘新乐校友（0603）于 2018 年发起设立，旨在为中国科学技术大学困难学生提供资助。该子基金的创立凝聚了众多 Google 校友的善意与贡献，为 USTCAF 的助学事业作出了重要贡献。

我们感谢所有参与和支持 Google 子基金的校友，包括历任管理者、捐赠者以及所有为基金运作提供帮助的志愿者。

## 关于治理接力的背景说明

2025 年 4 月，原负责人因离职，请求协助寻找继任者以完成治理交接。为确保基金在 Google 内部校友中良性传承，基金会管理层介入了寻找继任者的工作。2026 年 1 月 30 日后，原负责人改变了交接意向，相关工作目前已终止。

## 财务状况

截至 2026 年 2 月 2 日，Google 子基金余额为 **$115,879.18**。所有收支明细可在 [USTCAF 官网](https://www.ustcaf.org) 实时查询。

## 资金使用说明

USTCAF 遵循**专款专用原则**，在合规前提下充分尊重并遵循子基金负责人的意向。Google 子基金将严格锁定在设立初衷——资助中国科学技术大学困难学生。此前关于资源协同的讨论仅为理事会初步沟通，未涉及任何资金划拨。

此外，USTCAF 从未将 Google 子基金用于 K2S 项目。首届 K2S 经费由专项子基金与通用基金共同覆盖，后续计划引入企业赞助。

参考：[USTCAF 官网 Google Subfund 公告](https://www.ustcaf.org/about-us/governance/444-googler-subfund-01)

## 简介

我们建立这个 Github Repo 来收集整理 USTCAF 管理 Google 子基金的相关信息，并存档相关的来自各方文档。

这个Repo完全向公众公开，任何人都可以无障碍查阅里面的所有信息。如果任何人认为某些信息需要隐藏或者撤下，请当事人联系我们，或者在本repo直接提交 Github Issues。

本Repo希望模仿Wikipedia，不接收原创内容，所有内容需要来自其他信息平台。本仓库旨在促进透明度和问责制，而非针对个人。所有争议性话题的讨论均基于事实和文档，使用职务性称呼以保护隐私。
